# Frantic Castle

Frantic Castle is a web platform for the Frantic Castle mobile gaming studio. It gives players a place to explore game titles, follow studio updates, and stay connected with what the team is building.

## Tech Stack

- React 19
- TypeScript
- Vite 6
- Tailwind CSS v4
- Framer Motion

## Design

The interface was designed in Figma. View the design file [here](https://www.figma.com/design/TlHFlAFeikUGDjZ0OBPgWM/Frantic-Castle).

## Getting Started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## Project Structure

```
src/
  components/    Reusable UI components
  sections/      Page sections composed on each route
  pages/         Route-level page components
  constants/     Static content and data
  styles/        Global styles and animations
```
